### Video 3: Branch
<p>&nbsp;</p> 
Branch adalah jalur commit yang berada diluar dari jalur utama/master, dengan demikian, Branch dapat digunakan untuk membuat snapshot tanpa mengganggu jalur utama/master. Branching juga dapat digunakan utnuk melakukan kolaborasi dengan beberapa orang yang kemudian akan di merge untuk menyatukan proyek.
<p>&nbsp;</p> 

#### Membuat branch di Github
1. Edit file yang hendak di edit
2. Isi pesan commit
2. Tekan 'Create a new branch for this commit and start a pull request'

Branch yang telah dibuat kemudian dapat di merge dengan menekan 'Merge pull request'
